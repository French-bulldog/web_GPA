# scoreCalculationWeb
這是一個簡單的計算成績GPA網站

## 使用技術

- HTML
- CSS
- SCSS
- Javascript

## 如何使用

1. 從[這裡](https://github.com/French-bulldog/reCalculationWeb/)下載程式碼
2. 在瀏覽器中打開`index.html`文件
3. 開始使用！
